# 登录界面		

#### 	登录 

​			地址：/login            请求方式：POST

​			客户端发送数据

```
		{
			userAccount:"2016001",
            password:"123456",
            iemi:'1273197372173123128'   (判断手机新旧)
		}
```

​			服务器返回数据

```
		{
			code:"0",          (-1冻结  0登录成功  1登录失败)
			待定。。。。
		}
```
